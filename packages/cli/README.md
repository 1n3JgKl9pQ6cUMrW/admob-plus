# Commands

## `admob-plus info`

Get relevant version info about OS, toolchain and libraries

```
USAGE
  $ admob-plus info

OPTIONS
  -h, --help   show CLI help
  --clipboard  copy the environment report output to the clipboard

EXAMPLE
  $ admob-plus info
```
